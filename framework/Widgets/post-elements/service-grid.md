# Service Grid

This widget allows you to create customizable **Service Grid**. Navigate to the page you want to edit and open it in **Elementor edit mode**. Find the **CMF Service Grid** in the Elementor panel. Drag and drop it into the desired section of your page.

## Customize the Content

In **Content Tab** under **Layout** section select skin as your preference.
