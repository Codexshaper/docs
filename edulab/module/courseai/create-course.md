# Course Creation Wizards

## Create Course with AI

Navigate to **Admin → AI Tools → Create Course with AI** (or **Instructor → AI Tools → Create Course with AI**).

<!-- ![Create Course with AI Page](/assets/lms/images/module/courseai/create-course.png) -->

Fill in the form:

| Field | Description |
|-------|-------------|
| Topic | What the course is about (required) |
| Target Audience | Who should take this course |
| Difficulty Level | Beginner, Intermediate, or Advanced |
| Language | Language the course will be written in |
| Number of Chapters | How many chapters to generate (1–15) |
| Topics per Chapter | How many topics per chapter to generate (1–10) |
| Category | Assign the course to a category |
| Price Type | Free or Paid (sets the course pricing flag) |
| Instructors | Select which instructors to attach (admin/org only) |
| Extra Notes | Any additional context or style guidance for the AI |

Click **Generate Course**. The AI builds the full course structure — chapters, topics, requirements, outcomes, and FAQs — and redirects you to the course edit page when done.

<!-- ![Generating Progress](/assets/lms/images/module/courseai/create-course-progress.png) -->

<!-- ![Course Edit Page After Generation](/assets/lms/images/module/courseai/create-course-result.png) -->

> **Note:** Each topic is created as a Reading placeholder. Open the course edit page to add video, quiz, or other content to each topic.

## Text to Course

Navigate to **Admin → AI Tools → Text to Course**.

<!-- ![Text to Course Page](/assets/lms/images/module/courseai/text-to-course.png) -->

Paste any raw text — book chapters, lecture notes, articles, or transcripts (minimum 100 characters) — and select the language. Click **Convert to Course**.

The AI analyzes the content and returns a structured course outline. Once the result appears:

- Click **Use This** to save the draft outline for reference
- Click **Discard** to dismiss the result

<!-- ![Text to Course Result](/assets/lms/images/module/courseai/text-to-course-result.png) -->

> **Tip:** Use the generated outline as a blueprint — copy the chapter and topic names into your course manually, or use the **Create Course with AI** wizard for a fully automated workflow.
