# AI Usage Report

Navigate to **Admin → AI Tools → AI Usage Report**.

<!-- ![Usage Report](/assets/lms/images/module/courseai/usage-report.png) -->

| Section | Description |
|---------|-------------|
| Summary Cards | Total requests, today's requests, this week's requests |
| Feature Usage | Bar chart showing which AI tools are used most |
| Provider Breakdown | How many requests went to each AI provider |
| Audit Log | Paginated log of every AI action with feature, provider, user, and timestamp |

Use the filter bar to narrow results by feature, provider, or date range.

> **Note:** This page is visible to admins only.
