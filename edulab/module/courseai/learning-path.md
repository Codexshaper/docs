# Learning Path Builder

Students access the Learning Path Builder from their sidebar under **AI Tools → AI Learning Path**.

<!-- ![Learning Path Builder](/assets/lms/images/module/courseai/learning-path.png) -->

The student describes their learning goal (up to 500 characters) and clicks **Generate**. The AI returns a numbered, step-by-step learning plan with a title, description, and estimated time for each step.

Results can be copied to the clipboard with the **Copy** button.
